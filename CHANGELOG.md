# Changelog

All notable changes to the FABRK framework packages will be documented here.

This project uses [Changesets](https://github.com/changesets/changesets) for version management. Changelogs are automatically generated when versions are bumped.

## [Unreleased]

### @fabrk/core v0.1.0

- Initial release
- FabrkProvider with theme support
- useCostTracking, useCostBudget, useFeatureCost hooks
- Composable middleware system

### @fabrk/components v0.1.0

- Initial release
- 70+ production-ready UI components
- Built on Radix UI primitives
- Full TypeScript support

### @fabrk/ai v0.1.0

- Initial release
- Multi-provider AI toolkit (OpenAI, Anthropic, Google)
- Cost tracking and budget management
- Input/output validation with Zod

### @fabrk/design-system v0.1.0

- Initial release
- 18 terminal-inspired color themes
- ThemeProvider with dark mode support
- CSS custom properties and design tokens

### @fabrk/config v0.1.0

- Initial release
- Type-safe configuration builder
- Zod-powered validation
- Environment variable support

### create-fabrk-app v0.1.0

- Initial release
- Interactive CLI scaffolding tool
- 3 starter templates: basic, ai-saas, dashboard
- Automatic dependency installation and git init
