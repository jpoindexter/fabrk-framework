# FABRK Basic Usage Example

Minimal Next.js app demonstrating FABRK Framework packages.

## Setup

```bash
# From the monorepo root
pnpm install

# Run the example
cd examples/basic-usage
pnpm dev
```

## Packages Used

- `@fabrk/core` - Framework runtime, hooks, utilities
- `@fabrk/components` - UI components
- `@fabrk/design-system` - Themes and design tokens
