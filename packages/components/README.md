# @fabrk/components

70+ production-ready UI components for FABRK framework.

## Installation

```bash
npm install @fabrk/components @fabrk/design-system
```

## Usage

```tsx
import { Button, Card, Input } from '@fabrk/components'

export default function Page() {
  return (
    <Card>
      <Input placeholder="Enter text..." />
      <Button>Submit</Button>
    </Card>
  )
}
```

## Component Categories

- **UI Primitives** - Button, Input, Card, Badge, etc.
- **Forms** - Input, Select, Checkbox, Radio, etc.
- **Feedback** - Alert, Toast, Progress, etc.
- **Navigation** - Tabs, Accordion, Dropdown, etc.
- **Charts** - Area, Bar, Line, Pie charts
- **Data Display** - Table, DataTable, KPI cards

## Features

- Built on Radix UI primitives
- Terminal-inspired design system
- Full TypeScript support
- Accessible by default
- Tree-shakeable
