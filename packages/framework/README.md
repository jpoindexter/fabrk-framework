# fabrk

The React framework where AI agents are as native as pages and API routes.

Built on [Vite](https://vite.dev) with full Next.js App Router compatibility. Fork of [vinext](https://github.com/cloudflare/vinext) (MIT).

## Quick Start

```bash
npx create-fabrk-app my-app
cd my-app
fabrk dev
```
